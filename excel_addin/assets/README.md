# Icons

Place your add-in icons here with the following names:

- `icon-16.png` - 16x16 pixels
- `icon-32.png` - 32x32 pixels
- `icon-64.png` - 64x64 pixels
- `icon-80.png` - 80x80 pixels

## Requirements

- PNG format
- Transparent background
- Square dimensions
- Simple, recognizable design

## Design Tips

Use Domino branding:
- Domino logo
- Governance theme (shield, checkmark, etc.)
- Blue/green color scheme

## Tools

Create icons with:
- Figma
- Sketch
- Adobe Illustrator
- Canva (free)
